# Meal Planner & Household Grocery App

Repozytorium dokumentacji projektowej dla aplikacji webowej do wspólnego planowania posiłków, zakupów, liczenia kalorii i makroskładników oraz zarządzania domowymi wydatkami.

Dokumentacja została przygotowana jako kontekst wejściowy dla Codexa.

## Główne założenie

Aplikacja ma służyć kilku osobom należącym do jednego gospodarstwa domowego (`household`). Ma upraszczać:

- tworzenie i przechowywanie przepisów,
- planowanie posiłków,
- liczenie kalorii i makroskładników,
- generowanie list zakupów,
- zarządzanie zapasami,
- analizę wydatków,
- rozliczanie wspólnych zakupów.

Projekt powinien powstawać etapami. Pierwszym celem jest działające MVP, a nie pełny zestaw wszystkich funkcjonalności.
